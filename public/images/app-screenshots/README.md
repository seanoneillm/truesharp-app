# App Screenshots Directory

Place your app screenshots here with the following naming convention:

## Main App Screenshots (for carousel):
- `analytics-dashboard.png` - Analytics dashboard screenshot
- `calendar-analytics.png` - Calendar analytics view
- `games-screen.png` - Games/odds research screen  
- `marketplace.png` - Marketplace strategies view
- `seller-dashboard.png` - Seller dashboard view

## Feature Screenshots (for enhanced features section):
Place in `/feature-screenshots/` directory:
- `analytics-preview.png` - Analytics feature preview
- `sharpsports-sync.png` - SharpSports sync interface
- `strategy-details.png` - Strategy details modal
- `mock-betslip.png` - Mock sportsbook betslip
- `odds-research.png` - Odds research tools

## Image Requirements:
- Format: PNG preferred, JPG acceptable
- Dimensions: 375x812 (iPhone 13/14 dimensions) or similar mobile aspect ratio
- Quality: High resolution for retina displays
- Size: Optimized for web (< 500KB per image)
